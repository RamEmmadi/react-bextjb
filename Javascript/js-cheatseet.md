arr1.concat(arr2) : Return new array by joining arrays a1 and a2 together

```
const arr1 = [1,2,3,4]
const arr2 = [3,4,9,0]
console.log(arr1.concat(arr2)) //[1, 2, 3, 4, 3, 4, 9, 0]
```
